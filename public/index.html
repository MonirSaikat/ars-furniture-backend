ARS Furniture Apis.

Available routes:

  • Product routes:
    • Get: /products
    • Get: /products/:id
    • Post: /products { lable, price, imageUrl, rating } admin
    • Delete: /products/:id - admin
  • User routes:
    • Get: /users - admin
    • Post: /users { name, email, password } - admin
    • Delete: /users/:id - admin
  • Review routes:
    • Get: /reviews - admin
    • Post: /reviews { text, rating }
    • Delete: /reviews/:id
  • Auth routes:
    • Post: /auth/login { email, password }
    • Post: /auth/register { name, email, password }
  • Order routes
    • Get: /orders - admin
    • Post: /orders { discount, products: [] }
    • Put: /orders/:id/complete - admin
    • Put: /orders/:id/cancel
    • Delete: /orders/:id
